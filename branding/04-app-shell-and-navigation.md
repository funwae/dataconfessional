# Data Confessional – App Shell and Navigation Copy

## 1. Global App Elements

**App name / logo text:**

- "Data Confessional"

**Main navigation (side or top):**

- **Projects**

- **New Confession** (button)

- **Help** or **Guide**

If using per-project sub-navigation (tabs):

- **Overview**

- **Data Sources**

- **Dashboard**

- **Reports**

- **Q&A**

- **Exports**

---

## 2. Project List Screen

**Title:**

> Your confessions

**Subtitle (optional):**

> Each project is a quiet room where a specific question gets honest answers.

**Empty state text:**

> You don't have any confessions yet.

> Start by creating a project for a question you're trying to answer.

**New project button:**

- Text: **Begin a new confession**

**Project card fields:**

- Name

- Brief description

- Last updated

- Data sources count

- Status (Optional: "Just started", "In progress", "Briefing ready")

---

## 3. Project Overview Tab

**Title:**

> Project overview

**Sections:**

- **Question we're exploring**

- **What you brought in** (data sources summary)

- **What it's telling us so far** (highlight insights)

- **What you can do next** (actions: add data, generate dashboard, export)

**Empty state (no data yet):**

> This room is quiet.

> Add a few files or links and we'll start listening.

---

## 4. Data Sources Tab

**Title:**

> What you brought in

**Upload area heading:**

> Add files, links, or text

**Dropzone copy:**

> Drop CSV or Excel files here, or click to upload.

**URL input placeholder:**

> Paste a link you want the booth to read

**Text input placeholder:**

> Paste notes, transcripts, or any text you want analyzed

**Data sources list columns:**

- Name

- Type

- Status

- Last updated

- Actions (View summary, Remove)

**Processing status messages:**

- "Processing… listening closely."

- "Processed — this source is now part of the story."

- "Couldn't process this one. View details."

---

## 5. Dashboard Tab

**Title:**

> What the data is starting to say

**Primary CTA (if no charts yet):**

> **Generate first dashboard**

CTA helper text:

> We'll scan your tables and suggest charts that tell the clearest story.

**Chart card elements:**

- Title (e.g., "Revenue by region over time")

- Subtitle (insight):

  > "Confession: revenue in the West grew 28% while other regions were flat."

- Actions:

  - "Pin to top"

  - "Edit chart"

  - "Remove from dashboard"

**Empty state text:**

> No charts yet.

> Generate a first dashboard or add more data to give the booth more to work with.

---

## 6. Reports Tab

**Title:**

> Briefings and reports

**Secondary description:**

> This is where we turn what the data said into something you can say out loud.

**Primary CTA (if no report generated yet):**

- Button: **Create first report**

- Subtext:

  > Choose a format and audience, and we'll draft the sections for you.

**Report template names:**

- "Executive summary"

- "Sales overview"

- "Market snapshot"

- "General analysis"

**Section list labels:**

- "Executive summary"

- "Key findings"

- "Supporting evidence"

- "Risks and questions"

- "Next steps"

**Empty section copy:**

> This section is still blank.

> Ask the app to draft it from the data, or write your own.

---

## 7. Q&A Tab / Panel

**Title:**

> Interview your data

**Input placeholder:**

> Ask what the data has been hiding…

**Preset prompts:**

- "Give me the top three things I should know."

- "What changed the most since last period?"

- "Where are the biggest risks?"

- "What's surprising here?"

**Answer headings:**

- **Confession:** (direct answer)

- **Evidence:** (numbers, charts, sources)

- **Caveats:** (gaps, uncertainties)

**Example answer skeleton:**

> **Confession:**

> Mid-market accounts in the West are doing most of the heavy lifting, growing 32% while other segments are flat.

>

> **Evidence:**

> - Table: `q1_sales.csv` (columns: region, segment, revenue)

> - Chart: "Revenue by segment and region"

>

> **Caveats:**

> - No data for APAC after March.

> - 8% of deals have no segment tagged.

---

## 8. Exports Tab

**Title:**

> Briefings you can take into the room

**Description:**

> Turn this project into an executive summary, a deck, or a detailed report.

**New export button:**

- Text: **Create a new briefing**

**Export type labels:**

- "One-page executive summary (PDF)"

- "Slide deck (PPTX, 10–12 slides)"

- "Full report (PDF or DOCX)"

- "Markdown (for docs and wikis)"

**Existing exports list:**

- Columns:

  - Name

  - Type

  - Created

  - Actions (Download, Open)

**No exports yet state:**

> No briefings yet.

> When you're ready to share, create one from the latest findings.
